In this sprint we will be considering and refactoring entire codebases.

In this folder there are three implementations of an application.

1. A server side rendered application, where most work is done on the server
1. A client side rendered application, where most work is done in the browser
1. A hybrid application using a "hydration" approach, where the initial page load is server side rendered, but then the page is "hydrated" with client side JavaScript

Together, we will analyse these approaches to explain the tradeoffs between them.
